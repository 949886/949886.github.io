virtual关键字的作用

对指向子类对象的父类引用或指针，其调用同名方法（子类重写父类方法），声明virtual可以使其正确调用子类方法，否则就会调用父类方法。

include

using namespace std;

class Something
{
public:
Something() { cout<<”2”<<endl; }
~ Something() { cout<<”2”<<endl; }
};

class Parent
{
public:
Parent() { cout<<”1”<<endl; }
~ Parent() { cout<<”1”<<endl; }
};

class Child : public Parent
{
public:
Child() { cout<<”3”<<endl; }
~ Child() { cout<<”3”<<endl; }
protected:
Something _something;
};

int main()
{
Parent * ptr = new Child(); //Output “1231”.
delete ptr;

return 0;

}
