Quartz Core

图形上下文(Graphics Context)

图形上下文表示绘制目标。当程序需要绘制一个图形的时候，其首先要有一块画布以供绘制，Cocoa将其称为图形上下文，用CGContextRef表示。图形上下文中定义了基本的绘制属性，如颜色、裁减区域、线条宽度和样式、字体、混合模式等。

绘制图形

直线

若要设置绘制的颜色则可以调用UIColor的方法：

  • -(void)set;　　　　　　//设置Fill(填充)和Stroke(描边)的颜色
  • -(void)setFill;　　　　　//设置填充颜色
  • -(void)setStroke;　　　　//设置描边颜色

也可以调用纯C的方法：

  • CGContextSetFillColorWithColor(CGContextRef c, CGColorRef color);
  • CGContextSetStrokeColorWithColor(CGContextRef c, CGColorRef color);

二者是等价的。
